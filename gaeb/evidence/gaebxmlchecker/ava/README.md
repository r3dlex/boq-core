# GAEBXmlChecker AVA evidence

This directory is reserved for reviewed, human-local GAEBXmlChecker readiness
JSON for BVBS AVA X81, X84, and X86 fixtures.

- Do not commit checker binaries, archives, or executable payloads here.
- Do not let CI download or execute GAEBXmlChecker.
- A missing checker is recorded as `skipped_missing_checker`, not pass/fail.
- Evidence is readiness evidence only and is not official BVBS certification.

Schema and review rules: `docs/fixtures/gaebxmlchecker-ava-evidence.md`.
