# boq-core drafts

This directory holds work-in-progress drafts (ralplan, deep-interview
handoffs, exploratory specs) that are not yet ready to be promoted to
`.omc/plans/boq-core/` or `.omc/specs/boq-core/`.

## Conventions

- One draft per topic.
- Filenames: `ralplan-<topic>-draft.md`, `deep-interview-<topic>-draft.md`,
  or `<topic>-draft.md` for ad-hoc work.
- Promote to `plans/` or `specs/` only after architect + critic review.
- Do not store paid certification data, non-public AVA test material,
  or `obra backend` references here (ARCH-003).
